### 🛠️ What I build

- **Industrial IoT firmware** — ESP32 / SIM800L / SIM7080G, Modbus RTU & TCP, MQTT, OTA, offline buffering, watchdog patterns
- **PCBs & enclosures** — EasyEDA / KiCad, 2-layer → 4-layer, JLCPCB/PCBA, power-path design (BQ24074, TPS63802, MAX17048)
- **Production deployments** — energy & resource monitoring across substations, boilers, compressors, ETPs, generators, water meters
- **Teaching** — the **Full-Stack IoT Engineer Bootcamp** on YouTube, end-to-end: firmware → PCB → cloud → dashboard
